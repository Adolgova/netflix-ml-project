# Netflix ML Project

### Description
This project analyzes Netflix data and builds recommendation system using text features. It also includes a classification model to predict whether a title is movie or TV show.

### Pipeline
1. Data loading and cleaning  
2. Exploratory Data Analysis (EDA)  
3. Feature engineering (TF-IDF)  
4. Recommendation system (unsupervised learning)  
5. Classification model (supervised learning)  

### Result
Recommendation system based on content similarity  
Classification model with high accuracy
